# Differences From Other Languages

## Go
- Top level statements are legal
- No pointers, pointer types, indirection
- No init functions
- No labels, gotos
- No empty interfaces
- Any type can be a constant (except channels)
- Inclusion of 'any' type
- Optionals, Result types
